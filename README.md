
## React Vitrin Clone 

this project created with vite, React, typescript, nginx

## Installation

This project is dockerized

### development

run as follows:

```bash
cd './Docker/Dev'
docker-compose up --build
```
project is up and running on port 80

### Production

run as follows:

```bash
cd './Docker/Prod'
docker-compose up --build
```
project is up and running on port 80
